# Roadmap de implementacion

## Fase 1 - Base tecnica (completada)

- Monorepo configurado
- API y Web creadas
- Esquema Prisma inicial
- Login JWT inicial
- Dashboard base y vista de viajes

## Fase 2 - Operacion core

- CRUD de catalogos operativos
- Viajes con ciclo completo (alta, edicion, cierre, cancelacion)
- Gastos por viaje y utilidad

## Fase 3 - Taller e inventario

- Mantenimiento por unidad
- Uso de refacciones con descuento automatico
- Reporte mensual imprimible por unidad

## Fase 4 - Gobierno y analitica

- Auditoria completa
- Reportes operativos, tecnicos y financieros
- Exportacion PDF y Excel

## Fase 5 - Productivizacion

- Hardening de seguridad
- Pruebas e2e
- Observabilidad y despliegue
